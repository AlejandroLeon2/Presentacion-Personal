🧩 Proyecto: Página de Presentación Personal

📌Crea una página web de presentación mostrando su información personal, intereses y una pequeña galería.

✅ Requerimientos
Estructura HTML:

Usar !DOCTYPE html, html, head, body.

🔹 Incluir:
* header con el nombre
* section "Sobre mí"
* section "Hobbies"
* footer con contacto

🔹 Contenido:
* Nombre completo
* Foto personal o imagen representativa
* Lista de al menos 3 hobbies (ul o ol)

🔹 CSS:
* Cambiar tipografía (Google Fonts opcional)

🔹 Aplicar:
* Colores de fondo
* Márgenes y padding
* Centrar contenido principal

⚠️ Extra:
* Hover en los hobbies
